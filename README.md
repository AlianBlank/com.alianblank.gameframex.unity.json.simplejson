# 基于`SimpleJSON` 的二次修改

该库主要服务于 `https://github.com/AlianBlank/GameFrameX` 作为子库使用。


# 使用方式(三种方式)
1. 直接在 `manifest.json` 文件中添加以下内容
   ```json
      {"com.json.simplejson": "https://github.com/AlianBlank/com.json.simplejson.git"}
    ```
2. 在Unity 的`Packages Manager` 中使用`Git URL` 的方式添加库,地址为：https://github.com/AlianBlank/com.json.simplejson.git

3. 直接下载仓库放置到Unity 项目的`Packages` 目录下。会自动加载识别
